FICHIERS REQUIS DANS CE DOSSIER :

1. background4.bmp
   - Image d'arrière-plan pour le menu énigme
   - Format : BMP
   - Taille recommandée : 800x600 pixels

2. hover.wav
   - Son bref joué quand on survole un bouton
   - Format : WAV
   - Durée : 0.1-0.3 secondes

3. quiz_music.ogg  
   - Musique de suspense (battements de tambour)
   - Format : OGG
   - Joue en boucle pendant le quiz

4. Roboto-Regular.ttf
   - Police de caractères pour le texte
   - Vous pouvez télécharger depuis : https://fonts.google.com/specimen/Roboto
   - Ou utiliser n'importe quelle police .ttf

COMMENT OBTENIR CES FICHIERS :
- Pour la police : Téléchargez-la depuis Google Fonts ou copiez une police système
- Pour les sons/musique : Créez-les ou trouvez-les sur freesound.org
- Pour l'image : Créez-la avec Paint, GIMP, ou Photoshop (800x600)

NOTE : Le programme fonctionnera même si certains fichiers manquent, mais avec des erreurs.
